# **Music Theory \+ Computer Science Learning Journey**

**16-Week Evening & Weekend Program**

## **Timeline Overview**

**Time Commitment:** 8-12 hours/week

* **Weeknights:** 1-2 hours (theory, small experiments)  
* **Weekends:** 4-6 hours (main coding projects)

## **Phase 1: Audio Fundamentals \+ Simple Synth (Weeks 1-5)**

### **Week 1-2: Sound Basics & First Sounds**

**Weeknight Focus:**

* Read about sound waves, frequency, amplitude.  
* Watch Sebastian Lague's sound visualization videos.  
* Learn basic music theory (notes, octaves).

**Weekend Project:**

* Set up Python audio environment.  
* Generate your first sine wave.  
* Create a simple melody player.

**Theory Topics:**

* What is sound? (physics basics)  
* Digital audio (sample rate, bit depth)  
* Musical notes and frequency relationships

### **Week 3-4: Harmony Through Code**

**Weeknight Focus:**

* Learn intervals (minor 3rd, major 3rd, perfect 5th, etc.).  
* Study chord construction (triads, 7th chords).  
* Analyze chord progressions in modern music.

**Weekend Project:**

* Build "Harmony Explorer" tool.  
* Create chord generator functions.  
* Play and hear different chord types programmatically.

**Theory Topics:**

* Intervals and their emotional qualities.  
* Major, minor, diminished, augmented chords.  
* Chord progression basics.

### **Week 5: Mini-Project Week**

**Weekend Focus:**

* Build a playable keyboard synth.  
* Support multiple waveform types (sine, square, saw, triangle).  
* Implement a basic envelope (ADSR).

## **Phase 2: Visualization Magic (Weeks 6-10)**

### **Week 6-7: Waveform Visualization**

**Weeknight Focus:**

* Learn matplotlib or pygame basics for graphics.  
* Study oscilloscope principles.  
* Plan visualization architecture.

**Weekend Project:**

* Real-time waveform display.  
* Oscilloscope-style viewer.  
* Connect your synth from Phase 1 to the visualizer.

### **Week 8-9: Frequency Domain (Spectrum Analysis)**

**Weeknight Focus:**

* Learn FFT (Fast Fourier Transform) theory.  
* Study frequency spectrum concepts.  
* Watch tutorials on spectrum analyzers.

**Weekend Project:**

* Build a spectrum analyzer with frequency bars.  
* Frequency vs. amplitude visualization.  
* Color-coded frequency bands.

### **Week 10: Creative Visuals**

**Weekend Focus:**

* Lissajous curves (X-Y mode oscilloscope).  
* Circular/radial visualizers.  
* Particle systems responding to audio.

## **Phase 3: DAW World \+ Plugin Development (Weeks 11-14)**

### **Week 11-12: Cakewalk Deep Dive**

**Weeknight Focus:**

* Install and explore Cakewalk (DAW).  
* Learn basic MIDI concepts.  
* Study signal flow and routing.

**Weekend Projects:**

* Create a simple track using Cakewalk.  
* Learn basic mixing (levels, panning, EQ).  
* Use your Python synth to generate MIDI/audio for import.

### **Week 13-14: Plugin Development / Advanced Integration**

**Choose your path:**

* **Option A \- Python MIDI Tools:** Generate MIDI files programmatically or create algorithmic composition tools.  
* **Option B \- C\# / JUCE Plugin:** Set up the JUCE framework and build a simple gain/filter VST plugin.

## **Phase 4: Integration Project (Weeks 15-16)**

**Choose ONE capstone project:**

1. **Audio-Visual Performance Tool:** Real-time visualization system responding to live audio.  
2. **Intelligent Harmony Assistant:** Analyzes piano recordings and suggests complementary parts.  
3. **Custom Synthesizer \+ Visualizer:** Full-featured synth with a custom GUI and visualization panel.  
4. **Live Coding Music Environment:** Code-based music creation with real-time parameter control.

## **Resources & Sources**

### **Essential Libraries (Python)**

* **numpy:** Array manipulation and waveform generation.  
* **sounddevice:** Real-time audio I/O.  
* **scipy:** Signal processing (FFT, filters).  
* **librosa:** Advanced music and audio analysis.  
* **mido:** MIDI file handling.

### **Learning Resources**

* [Learning Music (by Ableton)](https://learningmusic.ableton.com/) \- Interactive theory basics.  
* [Think DSP (Allen Downey)](http://greenteapress.com/wp/think-dsp/) \- Python-based DSP intro.  
* [3Blue1Brown \- Fourier Transform](https://www.youtube.com/watch?v=spUNpyF58BY) \- Visual math explanation.  
* [The Audio Programmer (YouTube)](https://www.youtube.com/c/TheAudioProgrammer) \- C++ and Plugin development.

## **Tips for Success**

1. **Code along with tutorials:** Don't just watch; type the code to build muscle memory.  
2. **Keep a "Sound Journal":** Document what specific frequencies or waveforms sound like to you.  
3. **Connect with your partner:** Use her classical piano pieces as the "input data" for your experiments to see the harmony she plays visualized.
